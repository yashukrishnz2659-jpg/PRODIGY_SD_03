##Project Description

The Contact Management System helps users maintain a digital address book. It simplifies contact handling by providing features to manage personal or professional contacts in one place. The system reduces manual effort and improves accessibility of contact information.

##Technologies Used

Programming Language: Python

Data Storage: File handling / Database (depending on implementation)

Platform: Command Line / Console Based

##Features

Add new contacts

View saved contacts

Update existing contact details

Delete contacts

Search contacts by name or phone number

User-friendly interface

##Advantages

Easy to use and maintain

Saves time compared to manual contact storage

Organized and structured contact data

Suitable for small personal or academic projects


##Future Enhancements

Database integration (MySQL / SQLite)

Graphical User Interface (GUI)

Contact import/export functionality

User authentication system
